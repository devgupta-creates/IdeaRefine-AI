# IdeaRefine AI

IdeaRefine AI is a lightweight tool designed to help founders transform raw startup ideas into structured and actionable documentation.

The system captures an initial idea and guides users through a series of refinement questions related to the problem, users, solution type, and priorities. Based on these inputs, the system generates structured outputs such as Product Requirements Documents (PRD) and Business Requirements Documents (BRD).

The goal of this project is to explore how AI can support structured thinking and early-stage planning for startup ideas.

---

## Features

- Startup idea input interface
- Guided refinement through structured questions
- Automatic generation of PRD (Product Requirements Document)
- Automatic generation of BRD (Business Requirements Document)
- Simple and interactive web interface

---

## Tech Stack

- Python
- Streamlit
- OpenAI API
- LangChain
- MySQL
- Git & GitHub

---

## How It Works

1. User enters a startup idea.
2. The system asks a set of refinement questions.
3. User responses clarify the idea structure.
4. AI generates structured PRD and BRD documents.

---

## Project Purpose

Many early-stage startup ideas remain unclear or loosely defined.  
This project explores how a guided AI workflow can help structure ideas into clear documentation before execution.
